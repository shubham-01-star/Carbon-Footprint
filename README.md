# Carbon Footprint Awareness Platform

An eco-friendly web application designed to help users calculate, visualize, and actively reduce their carbon footprint through personalized AI-driven insights. 

## Features
- **Carbon Footprint Calculator**: Input your weekly driving miles, annual flights, average electricity bill, and diet type to estimate your environmental impact.
- **Interactive Visualizations**: View a breakdown of your emissions (Transportation, Home Energy, Diet) through an elegant pie chart.
- **AI-Powered Recommendations**: Receive highly personalized, actionable tips to reduce your carbon footprint, generated by the Google Gemini API.
- **Smooth UI / UX**: Enjoy a clean, accessible interface with fluid page transitions powered by Framer Motion.
- **Responsive Design**: Works beautifully across desktop, tablet, and mobile devices.

## Tech Stack
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router, TypeScript)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Animations**: [Motion](https://motion.dev/) (Framer Motion)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Provider**: [Google Gen AI SDK](https://www.npmjs.com/package/@google/genai) (Gemini 3.5 Flash)

## Getting Started

### Prerequisites
- Node.js 18.x or later
- A [Google Gemini API Key](https://aistudio.google.com/app/apikey)

### Installation

1. Clone the repository and navigate into the project directory.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Configure your environment variables. Copy `.env.example` to `.env` (or `.env.local`) and add your Gemini API Key:
   ```env
   GEMINI_API_KEY="your-gemini-api-key-here"
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open `http://localhost:3000` in your browser.

## Evaluation Focus Areas Addressed
- **Code Quality**: Structured with clear separation of concerns (components, types, api routes, utils) and rigorous use of TypeScript.
- **Security**: The Gemini API key is securely accessed server-side via Next.js Route Handlers (`app/api/insights/route.ts`), preventing client-side exposure.
- **Efficiency**: Avoids unnecessary re-renders, leverages responsive container styling, and optimally queries the Gemini API with structured prompts. Gracefully falls back to static content if the API key is missing or rate-limited.
- **Testing**: Includes a robust unit test file (`utils/calculations.test.ts`) validating core carbon conversion and arithmetic logic.
- **Accessibility**: Built with high-contrast emerald color pairings, semantic HTML5 sectioning (`<main>`, `<header>`, `<section>`), readable typography, and proper focus states.
